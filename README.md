
# CzechLibWrapper

Czech translation of LibWrapper version 1.13.3.0

## Popis
Tento modul poskytuje českou lokalizaci pro LibWrapper verze 1.13.3.0. Obsahuje překlady základních textů aplikace.

## Instalace
Pro instalaci modulu obvyklým způsobem můžeš použít následující odkaz ve Foundry VTT:

```
https://raw.githubusercontent.com/Lethrendis/CzechTranslationLibWrapper/main/module.json
```
